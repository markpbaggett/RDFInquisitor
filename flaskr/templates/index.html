About

RDF Inquisitor is a tool designed to help you question the RDF that your URIs point at. Unlike other tools, RDF Inquisitor assumes that your URIs adhere to the best practices described in definitive works like Best Practice Recipes for Publishing RDF Vocabularies, How to Publish Linked Data on the Web, and Cool URIs for the Semantic Web.

Checkout the project on GitHub for documentation and more information.
